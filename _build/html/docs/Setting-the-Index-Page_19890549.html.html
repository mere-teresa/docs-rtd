The Index Page is the page shown when the root index / is accessed.
For each siteacces you can configure IndexPage, as in eZ Publish 4.x

Put the parameter index_page in your ezpublish.yml file, under the right siteaccess category.

ezpublish.yml

Icon

If not specified, the index_page is the configured root content.

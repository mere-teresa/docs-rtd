eZ Publish Platform 5.<next> : Intro for Symfony developers¶

Added by gaetano.giunta@ez.no , edited by gaetano.giunta@ez.no on Jan 14, 2014

eZ Publish 5 is a distribution of Symfony2 similarly to symfony-standard with additional features to provide full CMS functionality. The CMS functionality is described elsewhere, this page and its sub page(s) aim to describe the Symfony features used and available out of the box.

Looking for documentation on Symfony itself? See Symfony Documentation at symfony.com.

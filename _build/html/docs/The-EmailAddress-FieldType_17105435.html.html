Description¶

The Selection FieldType stores an email address, which is provided as a string.

Validation¶

This FieldType``  uses the \ ``EmailAddressValidator validator as

resource, which will test the string supplied as input against a pattern, to make sure that a valid email address has been provided. | If the validations fail a ``ValidationError``  is thrown, specifying the error message.
